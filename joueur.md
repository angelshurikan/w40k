# Joueur
Ici se trouve la liste des instructions pour la fiche du personnage joueur.

## Fiche du personnage
- **Nom** : Le nom du personnage.
  - Modifiable par le joueur.
- **Âge** : L'âge du personnage.
  - Modifiable par le joueur.
- **Genre** : Le genre du personnage.
  - Modifiable par le joueur.
  - Homme ou femme
- **Vie max** : Le nombre de points de vie max du personnage.
    - Valeur initiale : 100
    - Non modifiable par le maître du jeu.
- **Vie actuelle** : Le nombre de points de vie actuels du personnage.
    - Valeur initiale : 100
    - Modifiable par le joueur.
- **Expérience** : Le nombre d'expérinces du personnage.
    - Valeur initiale : 0
    - Non modifiable par le joueur.
    - L'expérience (XP) est gagné quand on tue un monstre.
- **Expérience consomer** : Le nombre d'expériences consommé dans l'achat de compétences.
    - Valeur initiale : 0
    - Non modifiable par le joueur.
- **Niveau** : Le niveau du personnage.
    - Valeur initiale : 1
    - Non modifiable par le joueur.
    - Est calculé par l'expérience consommée par le joueur.
      - Niveau 1 : 500 XP
      - Niveau 2 : 1000 XP
      - Niveau 3 : 1500 XP
      - Niveau 4 : 2000 XP
